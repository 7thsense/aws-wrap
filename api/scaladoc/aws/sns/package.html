package sns

Access to Amazon Simple Notification Service (SNS).

Amazon Simple Notification Service is a web service that enables you to build distributed web-enabled applications. Applications can use Amazon SNS to easily push real-time notification messages to interested subscribers over multiple delivery protocols.

See the SNS object for available operations.
