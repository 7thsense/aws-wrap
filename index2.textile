h3. Amazon Web Service Library for Scala.

We support the following services:

 * SimpleDB
 * DynamoDB
 * S3
 * Simple Notification Service

To use it in any SBT project:

<pre><code>"aws" %% "dynamodb" % "0.1-SNAPSHOT"
</code></pre>

If you're in a Play 2.1 project, make sure to exclude Play from the dependencies:

<pre><code>"aws" %% "dynamodb" % "0.1-SNAPSHOT" exclude("play", "play_2.10")
</code></pre>
